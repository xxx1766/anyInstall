# Docker 安装脚本

一键安装 Docker（适用于 Linux）。

## 使用方法

```bash
curl -fsSL https://raw.githubusercontent.com/AnneBrown/anyInstall/main/docker/install.sh | sudo bash
```

或者手动下载执行：

```bash
curl -fsSL https://raw.githubusercontent.com/AnneBrown/anyInstall/main/docker/install.sh -o install.sh
chmod +x install.sh
./install.sh
```

## 说明

脚本使用 Docker 官方安装脚本（https://get.docker.com），安装过程大约需要 1-3 分钟。
